Logic Xử Lý Tổng Quan (Backend)
Kết nối Sheets: Sử dụng Google Sheets API (fetch JS) để read/write từ link mới: https://docs.google.com/spreadsheets/d/e/2PACX-1vQ-_-I6LLrifbZZPscBDUN9jufEyYrtf2tIIjtGihIScCU2tFp-HtuIgLkw6NqU0mUfOsEe9lIBTnIc/pub?gid=1944311512&single=true&output=csv.
Read: Load data ban đầu (danh sách chi/thu, tổng dư, mô tả list cho dropdown/settings).
Write: Append dòng mới cho chi/thu, update tổng kết.
Auth: OAuth cho iOS (prompt user login Google).
Xử lý Data:
Chi: List tạm (array JS), tính tổng, mô tả combine (checkboxes + dropdown + text).
Thu: Tương tự, đơn giản.
Tổng kết: Tính SUM(J) - SUM(D) cho dư lý thuyết, prompt nhập thực tế, ghi chênh lệch.
Sync: Data realtime giữa 2 device qua sheet chung.

Logic Giao Diện (Xử Lý Event)
Nhập số: On input, validate số.
Nút +: Push to array tạm, render list clickable, clear input.
Click số trong list: Populate input, đổi nút thành V (confirm), update on confirm.
Nút Xóa hết: Clear array, reset UI.
Mô tả: On change checkboxes/dropdown/text, combine string.
Nút Thêm: Call Sheets API append, show toast success, reload data.
Settings: Modal popup: Checkboxes chọn 8 mô tả, save local/sheet.
Tổng kết: Read sheet, prompt nhập thực tế, calculate, append.
